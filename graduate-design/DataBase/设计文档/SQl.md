### 用户总表

| userId       |         用户ID（主键，不能为空）         |
| ------------ | :--------------------------------------: |
| userNickName |           用户昵称（不能为空）           |
| userPassword |           用户密码（不能为空）           |
| role         | 用户角色（分为 admin 和 user，不能为空） |
|              |                                          |



### 普通用户注册。

1. 注册的时候只需要填写**用户ID** **用户名**  **密码** ，以及**确认密码** 。
2. 在购物的时候，用户需要添加收获地址。（向数据库中插入一条数据）。
